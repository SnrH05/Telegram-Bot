# Repoyu Kaydetme

Bu workflow değişiklikleri commit'leyip GitHub'a push eder.

## Adımlar

// turbo-all

1. Tüm değişiklikleri ekle:

```powershell
git add -A
```

2. Commit at (commit mesajını belirt):

```powershell
git commit -m "<COMMIT_MESAJI>"
```

3. GitHub'a push et:

```powershell
git push
```

## Kullanım

`/save-repo` yazarak bu workflow'u çalıştırabilirsiniz. Commit mesajını belirtmeniz gerekir.
